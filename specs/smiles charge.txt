To run a Graph Neural Network (GNN) for pKa, you do not need a GPU. These models are lightweight and will run in seconds on your CPU because the "graph" of a small molecule like Flumazenil or a Ferulate is tiny compared to something like an image or a large language model.
The Package

You need pka-predictor-moitessier. It is the most direct implementation of the Moitessier Lab's GNN pKa model.
The Setup

Run this in your terminal:
Bash

pip install pka-predictor-moitessier
